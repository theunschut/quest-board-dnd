<objective>
Show the active group name and a switch-group link in the navigation.

Purpose: UX-05 requires the nav to display the current group and let the user return to the picker. The group name was stored in session by `GroupPickerController` (plan 30-01) under `SessionKeys.ActiveGroupName`; this plan reads it in both layouts and renders a dropdown item that links to the picker. Storing-then-reading the name avoids a per-request DB lookup (D-16 resolution: session storage).

Output: a group-switch item in the authenticated user dropdown of `_Layout.cshtml` (between Profile and Logout) and the equivalent item in `_Layout.Mobile.cshtml`.

Implements D-14, D-15, D-16, D-18 — covers UX-05.
</objective>
