## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client → controller POST | Browser-originated state-changing requests must carry a valid anti-forgery token so a cross-site page cannot forge them |
| app → Resend API (external HTTP) | Upstream 429 rate limits must degrade gracefully; retry budget must be bounded |
| deleted-DM data → reminder job | A quest whose DungeonMaster row was removed must not crash the background job |

Attributes verified already present today. |
| T-34.1-04 | Denial of Service | Resend stats fetch retry loop (ASVS V7) | mitigate (test) | Task 1 Test B asserts the retry budget is bounded to 4 total attempts, proving no unbounded retry storm against the upstream API. |
