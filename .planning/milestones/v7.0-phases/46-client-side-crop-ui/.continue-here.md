## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Bash tool CWD drift into a worktree directory | A `cd "...worktrees\agent-X" && pwd` call left the Bash tool's cwd inside the worktree for all subsequent calls in the session (cwd persists across Bash calls). The next worktree-merge attempt then ran `git merge-base`/`gsd-sdk query worktree.cleanup-wave` from inside the wrong worktree, producing a confusing false "base_mismatch" failure. | advisory | Before any worktree merge/cleanup command, verify `pwd` and `git rev-parse --abbrev-ref HEAD` resolve to the primary worktree root and the phase's working branch. If you ever `cd` into a worktree path to inspect something, immediately `cd` back (or better, use `git -C <path> ...` instead of `cd` at all). |
| Raw `node -e '...'` / `grep -h` via Bash tool produce mangled output on this machine | An active `rtk` (Rust Token Killer) CLI proxy hook rewrites some shell commands transparently. A `grep -h ...` call was rewritten into an `rtk grep` invocation with incompatible flags and printed rtk's own help text instead of results. Separately, a `node -e '...'` one-liner constructing a JSON string had its escaped backslashes silently stripped before node ever saw them (path became `C:Reposquest-board...` instead of `C:\Repos\quest-board\...`). | advisory | Prefer the dedicated Grep/Write/Edit tools over shelling out to `grep`/`node -e` for anything involving path escaping or JSON construction. Use Bash only for plain filesystem/git/dotnet commands. |
| `/tmp/...` paths from Bash's `mktemp` don't resolve for Windows-native tools | Git Bash's `/tmp` maps to a real Windows path (e.g. `C:/Users/.../AppData/Local/Temp`), but native Windows executables invoked via Bash (like `node.exe`) interpret `/tmp/x` literally as `C:\tmp\x`, which doesn't exist. | advisory | Resolve the real Windows path first (`cd /tmp && pwd -W`) before handing a temp path to any non-bash-native tool, or just avoid raw temp files for JSON manifests ‚Äî use Write/Read/Edit tools instead, which handle path translation correctly. |

**Severity values:** `blocking` ‚Äî resuming agent must pass understanding check before proceeding. `advisory` ‚Äî important context, does not gate resumption.
