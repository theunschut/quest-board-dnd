<objective>
Change the desktop Guild Members page (`QuestBoard.Service/Views/GuildMembers/Index.cshtml`) from a two-column side-by-side layout to two vertically stacked full-width sections: "My Characters" on top, "Guild Roster" directly below.

Purpose: The Guild Roster section grows as the group gains characters, but the current `col-md-6` layout constrains it to half the page width, wasting horizontal space and forcing a cramped card grid. Stacking the two sections vertically lets the roster use the full page width, so its `character-grid` (which already uses an auto-fill responsive grid) flows across more columns as it grows.

Output: A modified `Index.cshtml` with the Bootstrap `row`/`col-md-6` grid replaced by two full-width stacked card blocks. No ViewModel, controller, route, or CSS changes. The mobile variant and all card styling remain untouched.
</objective>
