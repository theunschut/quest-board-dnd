## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| request → ActiveGroupId resolution | A regular authenticated request that unexpectedly has null ActiveGroupId must fail fast, not silently scope wrong/all data (ASVS V4 Access Control) |
| EF Core query filter → group-scoped data | A non-existent/incorrect ActiveGroupId must return empty, never another group's data |
| DB int enum values → domain enums | AutoMapper casts must not silently wrap/truncate an undefined int into a wrong enum value |
| Hangfire job → transient upstream failure | Retry policy must be bounded and correctly configured |
